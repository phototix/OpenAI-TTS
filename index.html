ElevenLabs Text-to-Speech Extension

A Chrome extension that reads selected text using ElevenLabs API

About the Extension

This Chrome extension allows you to select text on any webpage and have it read aloud using the high-quality ElevenLabs text-to-speech API.

Try selecting this text or any text on this page, then right-click and choose "Read Aloud" from the context menu.

The extension will split the text into chunks and read it sequentially, showing a popup with the currently spoken text.

You'll need to configure your ElevenLabs API key and preferred voice ID in the extension options for it to work properly.

How to Install

  1. Download the extension files
  2. Open Chrome and go to chrome://extensions
  3. Enable "Developer mode"
  4. Click "Load unpacked" and select the extension folder
  5. Configure your API key in extension options
  6. Start using the extension by selecting text and right-clicking

Features

  • Right-click context menu integration
  • Configurable chunk size for text processing
  • Popup display showing current text being read
  • Easy configuration panel

Test the Extension

Select some text in the area below or anywhere on this page, then right-click and choose "Read Aloud" from the context menu.

Try selecting text with different lengths to see how the chunking works. You can configure the chunk size in the extension options.
